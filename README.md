# 🦾 Wireless Bionic Hand

## Overview

This project presents a wireless bionic hand capable of replicating finger movements using flex sensors and an ESP32-based control system.

Finger bending is detected using flex sensors worn on a glove. The sensor data is transmitted wirelessly to an ESP32, which actuates servo motors to reproduce the corresponding finger movements on the robotic hand.

The project combines mechanical design, embedded systems, wireless communication, and CAD modelling to demonstrate a low-cost robotic prosthetic concept.

## Why This Project?

Robotic prosthetics have the potential to improve accessibility and assistive technologies by mimicking natural human hand movements.

This project explores a simple and cost-effective approach to wireless gesture-controlled robotic hands using flex sensors, servo motors, and an ESP32 microcontroller.
